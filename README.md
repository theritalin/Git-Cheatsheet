# Git-Cheatsheet


- Clone the repository:

```bash
  git clone "address"
```

- Pushing to Github repository:

```bash
  git add .
  git commit -m "MESSAGE"
  git push -u origin main
```


- Creating new repository:

```bash
  git init
  git add .
  git commit -m "first commit"
  git remote add origin "ADDRESS"
  git push origin main
```
